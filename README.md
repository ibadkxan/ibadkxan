## 📌 Featured Projects

### ⚡ [VoltSlot — EV Charging Slot Booking System](https://github.com/ibadkxan/ev-charging)
> Python · Flask · MySQL · REST API · Session Auth

Full-stack EV charging slot booking platform with separate user and admin dashboards. Features real-time slot availability checks, wallet-based payments, and 5 MySQL triggers handling booking automation, double-booking prevention, and wallet transactions. Schema designed in 3NF with views and indexes.

---

> React Native · Android · JavaScript · Database Backend

Android app built for university lost & found management. Users can report or search for items with images; the backend matches submissions against existing records using security questions and item details to flag potential matches. Built with React Native after initially prototyping as a web app.
